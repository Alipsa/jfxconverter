This package provides several utility geometry classes. It contains :
  • a Polygon2D class, using float coordinates instead of the int coordinates of the java Polygon class, but having the same characteristics
  • a Polyline2D class, allowing to defined a polyline without having to define a set of separate lines
  • a Vector2D class, containing many methods useful to perform 2D Vector calculations
  • a ShapeNormalizer class, allowing to convert Shapes to a set of Arcs, Polygons, and Lines
  • a GeometryRuler class, allowing to calculate the area and length of Shapes
  • a ClippableShape class, allowing to perform a intersect method on a shape which is not an Area
