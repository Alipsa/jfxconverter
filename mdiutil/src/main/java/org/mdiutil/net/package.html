This package contains two classes which can be used to connect to URLs nested inside a Jar file.

  • The {@link org.mdiutil.net.NestableURLConnection} is a connection which can point to an URL nested inside archives in an archive file
  • The {@link org.mdiutil.net.JarClassLoader} is a class loader which can load class files nezsted inside inside archives in an archive file

This package uses heavily classes from the java.net package and associated private classes, all under the LGPL license and coming from the OpenJDL project.
