Drivers

Several specific drivers are proviced with the project:
  • The jfxConverter−ppt.jar contains a PowerPoint Graphics2D implementation
  • The jfxConverter−svg.jar contains a SVG Graphics2D implementation
  • The jfxConverter−eps.jar contains an EPS Graphics2D implementation
Note that in general the JavaFX content must be shown in order to be converted. However you don't need to be in the FX Application Thread to execute the conversion.

Using the PPT driver

Main Article: PPT driver

You should use the jfxConverter−ppt.jar in addition to the jfxConverter.jar.

Using the SVG driver

Main Article: SVG driver

You should use the jfxConverter−svg.jar in addition to the jfxConverter.jar.

Using the EPS driver

Main Article: EPS driver

You should use the jfxConverter−eps.jar in addition to the jfxConverter.jar. Note that the PowerPoint driver uses any version of the jlibeps library.
