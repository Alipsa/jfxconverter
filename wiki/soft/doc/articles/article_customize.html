Adding a footer and header

The most simple way to customize the wiki presentation is to add a footer and header file to each article. It is possible to add some custom content in the footer in the configuration file.

Setting a custom background for the articles

Main Article: Setting a background

It is possible to set a custom background image or color for the articles of the wiki.

Using properties

It is possible to configure content in the wiki by using properties. Properties allow to define key / values for string replacements in the text.
