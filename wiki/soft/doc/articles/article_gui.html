GUI interface

Double clicking on the Jar file of the application without specifying any command-line parameter will open the GUI of the application.


Configuration

Many of the configuration properties wich are available through the command-line or the configuration file can be specified directly through the GUI. Additionally:
  • Named properties can be set through the "Property / Values" file parameter
  • Additional configuration properties can be set through the "Additional Configuration" file parameter
